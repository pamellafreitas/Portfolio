Sobre Mim

Olá, sou Pamella, uma entusiasta do desenvolvimento front-end e do design UI/UX, dedicada a explorar os limites da criatividade digital. Aos 26 anos, estou imersa em estudos e projetos que alimentam minha paixão por tecnologia e design.

Com uma formação sólida em ballet clássico, adquiri valores como disciplina, expressão artística e atenção aos detalhes, habilidades que agora aplico em minha jornada no mundo do desenvolvimento. Estou em constante evolução, buscando expandir meu conhecimento e habilidades para oferecer soluções digitais impactantes.

Além de meu compromisso com o desenvolvimento front-end e o design UI/UX, também ofereço meus serviços como freelancer, buscando oportunidades de colaborar em projetos desafiadores e inspiradores. Estou determinada a crescer nesta área e ansiosa para explorar novas oportunidades e desafios.

Advogada

O projeto foi desenvolvido utilizando HTML5, CSS3, JavaScript e Bootstrap para criar uma experiência web interativa e responsiva. A utilização da Intersection Observer API permitiu animações suaves, enquanto o Bootstrap simplificou o desenvolvimento com seus componentes pré-estilizados e sistema de grid. O resultado é um site moderno, funcional e visualmente atraente.

Explore o Projeto

Pizzaria

Neste projeto, empreguei HTML5, CSS3 e JavaScript para criar uma página web responsiva para um restaurante fictício. O HTML5 foi responsável pela estruturação do conteúdo, o CSS3 pela estilização e layout, enquanto o JavaScript adicionou funcionalidades interativas, como rolagem suave e recolhimento automático do menu em dispositivos móveis. Essas tecnologias foram combinadas para proporcionar uma experiência de usuário fluida e consistente em uma variedade de dispositivos.

Explore o Projeto

Beauty Salon

Este projeto, desenvolvido durante o curso da Rocketseat, é um site para um salão de beleza chamado Beauty Salon. Utiliza tecnologias como HTML, CSS e JavaScript para criar uma experiência interativa e responsiva para os usuários. O site apresenta seções como "Início", "Sobre", "Serviços", "Depoimentos" e "Contato", permitindo que os visitantes conheçam os serviços oferecidos, leiam depoimentos de clientes e entrem em contato com o salão.

Explore o Projeto

Clima Tempo

Este projeto do Clima Tempo foi desenvolvido como parte do curso "Hora de Codar". É uma aplicação web simples que permite aos usuários consultar o clima atual de qualquer cidade. Utilizando a API do OpenWeatherMap, o projeto faz chamadas assíncronas para obter os dados meteorológicos e os exibe de forma clara na interface do usuário.

Explore o Projeto

Portfólio

No desenvolvimento do meu portfólio pessoal, optei por uma abordagem utilizando HTML, CSS e JavaScript para garantir uma experiência de usuário intuitiva e responsiva. Utilizei técnicas avançadas de CSS, incluindo animações e flexbox, para criar uma interface visualmente atraente e dinâmica. Além disso, implementei um menu responsivo para garantir uma navegação fluida em dispositivos móveis e desktops.
